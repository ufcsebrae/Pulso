Visão Geral da Execução

Este painel consolida os dados financeiros de 2025, analisando a relação entre o Valor Planejado e o Valor Executado. Para uma análise mais clara, os projetos foram divididos em duas categorias:

Projetos Exclusivos

São projetos cujo orçamento e execução pertencem a uma única unidade de negócio. A performance aqui reflete diretamente o desempenho daquela unidade específica.

Projetos Compartilhados

São projetos que envolvem duas ou mais unidades, com orçamento e execução distribuídos entre elas. A análise desses projetos requer uma visão colaborativa.
